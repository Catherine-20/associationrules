# Association Rules Example
INGABIRE CATHERINE-671041
This project demonstrates Association Rule Mining using the `mlxtend` library in Python.

## Description
- Uses a small sample dataset representing shopping baskets.
- Each transaction lists items purchased together.
- Finds frequent itemsets using the Apriori algorithm.
- Generates association rules to show which items are likely bought together.
